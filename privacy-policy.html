Party Mahjong Table Privacy Policy

Last updated: 2026-07-17

English

Party Mahjong Table is designed for in-person play. The app does not require an account and does not use advertising, analytics SDKs, tracking SDKs, public room listings, stranger matchmaking, or developer-operated game servers. It communicates only on the host device and the same local network.

Data the App Handles

  • Player display names entered in the app or browser.
  • Seat selection, tile state, discard records, exposed tiles, flower tiles, turn state, and local action log.
  • Local network addresses shown by the host device so nearby devices can join.

Storage and Transmission

  • The host device is the authoritative source of table state.
  • Table state can be stored locally on the host device so the table can recover after interruption.
  • When browser join or app join is used, the host sends each player only that player's visible snapshot over the local Wi-Fi or personal hotspot.
  • Local rooms do not upload rounds, player names, tiles, logs, or network addresses to a public server controlled by the developer.
  • The app does not upload rounds, player names, tiles, logs, or network addresses to a developer-controlled server.

Tracking, Advertising, and Accounts

The app does not track users across apps or websites. It does not contain third-party advertising or analytics SDKs. It does not require account registration.

Subscription and Money-Related Table Features

The free edition allows the host to complete up to 5 successful deals per local day. The allowance is stored only on that device and is not uploaded. The optional auto-renewing PRO Monthly subscription provides unlimited starts plus Classic Chinese and Japanese realistic tile designs while active. It renews monthly unless canceled and can be managed or canceled in the App Store; after cancellation, access continues through the end of the current subscription period. Reaching the free limit or ending a subscription never interrupts a round already in progress. Browser participants do not need to purchase or subscribe. The app provides no payments tied to scores or outcomes, money-related table features, scoring, result settlement, publicly listed rooms, or stranger matchmaking.

Local Network Permission

The app asks for Local Network access so nearby devices on the same Wi-Fi or personal hotspot can join the host's local table through the iOS app or a browser.
