Party Mahjong Table Support

Last updated: 2026-06-30

English

Party Mahjong Table is an in-person local-network mahjong table utility. It does not require an account and communicates only on the host device and the same Wi-Fi or personal hotspot.

Support Contact

Public support contact: pocket_consulting@163.com.

Response language: Simplified Chinese, Traditional Chinese, English, or Japanese.

Common Setup Help

  • For local rooms, all players must be on the same Wi-Fi or personal hotspot.
  • The host device creates the local room and starts the local web service.
  • Other devices can join through the iOS app or by opening the local browser address shown by the host.
  • If the local browser page does not open, check Local Network permission and confirm that all devices are on the same local network.

Privacy

The app does not provide accounts, ads, tracking, analytics SDKs, public rooms, stranger matchmaking, scoring, settlement, or payments tied to outcomes. Hosts can start 5 rounds free per day. The optional non-consumable Unlimited Tables Forever purchase unlocks unlimited starts plus classic realistic tiles; browser participants never need to purchase it.

Privacy policy: Party Mahjong Table Privacy Policy.
