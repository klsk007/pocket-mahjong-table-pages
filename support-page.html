Pocket Mahjong Table Support

Last updated: 2026-06-24

English

Pocket Mahjong Table is a local-network table utility for in-person mahjong gatherings. It does not require an account and does not use a public game server.

Support Contact

Public support contact: pocket_consulting@163.com.

Response language: Simplified Chinese, Traditional Chinese, English, or Japanese.

Common Setup Help

  • All players must be on the same Wi-Fi or personal hotspot.
  • The host device creates the room and starts the local web service.
  • Other devices can join through the iOS app or by opening the local browser address shown by the host.
  • If the browser page does not open, check Local Network permission and confirm that all devices are on the same local network.
  • The app does not upload rounds to a public server. The host device is the table state source.

Privacy

The app does not provide accounts, ads, tracking, analytics SDKs, public rooms, stranger matchmaking, payments, scoring, or result settlement.

Privacy policy: Pocket Mahjong Table Privacy Policy.
